## Qu'est que le InfinAdd ?

Le InfinAdd est un additionneur de nombres infinis.
Ce calculateur permet de faire des additions de nombres négatifs et infinis.

### Compétences travaillées

- Manipulation des différents types de variables et convertions de type
- Manipulation de nombre infini et négatif

### Installation

Pour compiler ce projet, il vous suffit de vous rendre a la racine et de lancer la commande :

```bash
make
```

## Utilisation

Une fois le binaire généré, il vous suffit de le lancer grace a la commande suivante :

```bash
./infinadd [nbr1] [nbr2] ...
```
